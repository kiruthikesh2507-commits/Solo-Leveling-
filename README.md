# AWAKEN SYSTEM
## Real-Life RPG Self-Improvement App

---

### HOW TO RUN LOCALLY (like a desktop app)

**Option 1 — Python (easiest, no install needed):**
1. Open a terminal/command prompt in this folder
2. Run: `python -m http.server 8080`
3. Open Chrome/Edge and go to: `http://localhost:8080`
4. Click the install icon (⊕) in the address bar to install as desktop app

**Option 2 — Node.js:**
1. Run: `npx serve .`
2. Open the URL shown, then install via browser

**Option 3 — VS Code:**
Install the "Live Server" extension, right-click index.html → "Open with Live Server"

---

### INSTALL AS DESKTOP/MOBILE APP (PWA)

**Chrome/Edge Desktop:**
- Open the app in browser
- Look for install icon (⊕) in address bar
- Click "Install" — it opens like a native app

**Android:**
- Open in Chrome → tap ⋮ → "Add to Home Screen"

**iOS:**
- Open in Safari → tap Share → "Add to Home Screen"
